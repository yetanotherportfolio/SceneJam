# SceneJam

Simple, engine agnostic, 2d scene editor.

Support:
 - placing elements in a 2d scene
 - configure sprite animations
 - configure tweens
 - setup game width/height
 - setup scene background color

## RUN
using [NW.js](https://nwjs.io/)

$ nw .

## DEBUG

$ npm run debug
